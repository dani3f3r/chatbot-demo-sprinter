Asistente de voz · Demo de reservas telefónicas

Habla con el asistente de reservas del Grupo Con Fuego.

Esta demo reproduce una llamada real a un asistente de voz que gestiona las reservas de los restaurantes del grupo.

Para probarlo, utiliza el botón redondo de teléfono azul situado en el panel de la derecha.

  • El navegador te pedirá permiso para usar el micrófono.
  • Habla con el asistente como si llamaras al restaurante.
  • Tras colgar, puedes iniciar una nueva llamada desde el mismo botón azul.

Recomendado usar auriculares y un entorno silencioso para una mejor experiencia.

Cómo usar esta demo

En este panel encontrarás el botón de teléfono azul que te permite iniciar y finalizar la llamada con el asistente de reservas.

  • Haz clic en el teléfono azul para iniciar la llamada.
  • Habla con el asistente y realiza una reserva de prueba.
  • Vuelve a pulsar para colgar cuando quieras terminar.

Puedes repetir la prueba tantas veces como quieras para ver cómo responde el asistente en diferentes situaciones.
